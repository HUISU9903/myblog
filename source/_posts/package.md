# 패키지의 설치 방법

패키지는 **라이브러리**라고도 하며, Rstudio에 내장되어 있는 내장함수만이 아닌 

다른 함수 들도 사용 할 수 있게 해주는 귀중한 역할을 한다.

패키지는 한번 설치하면 이후에는 설치하는 과정은 필요 없지만 

파일을 열때 마다 불러오는 과정은 필요하다.

```r
install.packages("패키지 이름") # 패키지를 설치
```

```r
library(패키지 이름) # 패키지를 불러옴
```
